# Task1 - Smart Calculator (Java)
This project is part of the **ElevatedLabsWorks** collection and implements a **Smart Calculator** using object-oriented programming principles in Java.

## 🧠 Features
- Basic arithmetic operations (Addition, Subtraction, etc.)
- Uses inheritance, abstraction, and method overriding
- Maintains a simple calculation history
- Console-based user interaction

## 🧩 Classes Overview

| Class                | Description                               |
|----------------------|--------------------------------------------|
| `CalculatorBase`     | Abstract class defining method signatures  |
| `Calculator`         | Implements basic calculator logic          |
| `AdvancedCalculation`| Extends `CalculatorBase` with enhancements |
| `CalculationHistory` | Stores and prints operation history        |




- Designed a calculator system using **object-oriented design**
- Created a **base class (`CalculatorBase`)** with abstract methods
- Implemented basic operations in a concrete class (`Calculator`)
- Built an **advanced version** using `AdvancedCalculation` which:
  - Extends the base calculator
  - Adds functionality to **store and show calculation history**
- Separated history tracking into its own class `CalculationHistory` to follow **single responsibility principle**

The purpose was not just to compute values, but to **build a clean, extensible, and maintainable Java codebase**.
